# Retail Sales Analysis and Revenue Prediction

## Overview

This project analyzes retail sales data and predicts future revenue using Machine Learning.

## Technologies Used

* Python
* Pandas
* Matplotlib
* Scikit-Learn

## Features

* Data Analysis
* Statistical Summary
* Correlation Analysis
* Sales Trend Visualization
* Revenue Prediction using Linear Regression

## Dataset Attributes

* Sales
* Customers
* Advertising Spend

## Machine Learning Model

Linear Regression

## Key Findings

* Higher customer count increases sales.
* Advertising positively impacts revenue.
* Sales show a steady upward trend throughout the year.

## Results

The model successfully predicts future sales based on customer and advertising data.

## Future Scope

* Use larger retail datasets.
* Build dashboards using Power BI or Tableau.
* Deploy the model using Flask or Streamlit.
